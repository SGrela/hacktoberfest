HacktoberFest

Lista de proyectos de la UGR que participan en la HacktoberFest, donde por colaborar y hacer un número determinado de pull requests te regalan una camiseta.

Haced un fork de este repositorio para añadir vuestro proyecto; haced un pull request cuando lo hayáis hecho. Previamente, elige o añade una serie de issues a tu proyecto etiquetándolos Hacktoberfest, que son los únicos elegibles para la camiseta.
